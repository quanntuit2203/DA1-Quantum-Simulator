Documents

Commands Explain
reset(num_qubits) Set up a quantum simulation, with num_qubits qubits.
write(value, target_qubits) Write the specified value to the target qubits.
not_gate(target_qubits) Perform a logical NOT operation on the qubits specified by target_qubits
cnot_gate(target_qubits, condition_qubits) Perform a logical CNOT operation on the qubits specified by target_qubits, using condition_qubits as the condition
hadamard(target_qubits) Perform a HADAMARD operation on the qubits specified by target_qubits
chadamard(target_qubits,condition_qubits) Perform a CHADAMARD operation on the qubits specified by target_qubits, using condition_qubits as the condition
swap(qubit1, qubit2, condition_qubits) Swap two qubits specified by qubit1 and qubit2, using condition_qubits as the condition
phase(theta_degrees,target_qubits) Perform a Z rotation (phase), by the specified angle.
Important note: If more than one qubit is specified, this will perform a conditional phase shift, using all specified qubits (like a CZ gate).
rotatex(theta_degrees,target_qubits) Rotate in X rotation
rotatey(theta_degrees,target_qubits) Rotate in Y rotation
